## install typescript globally

```

  npm i -g  typescript

```

> check tsc version

```
tsc --version
```

with this we can use tsc command

we can use:

```
tsc --init
  or
npx tsc --init
```

> command to run ts file

```
tsc filename.ts
```

** with this the file will generate a .js file which will work **
